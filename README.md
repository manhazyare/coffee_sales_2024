# coffee_sales_2024
# ☕ Coffee Sales Analytics Project

Welcome to the Coffee Sales Analytics Project repository! 🚀

This project explores transactional coffee sales data to uncover actionable business insights — covering sales trends, product performance, customer behavior, and revenue patterns. Designed as a portfolio project, it highlights practical skills in data analytics and SQL-based reporting.

---

## 📖 Project Overview

This project analyzes a single dataset of coffee shop transactions spanning **March 2024 – March 2025**, containing **3,636 records** with the following fields:

| Column | Description |
|---|---|
| `date` | Transaction date |
| `datetime` | Full timestamp of the transaction |
| `hour_of_day` | Hour the transaction occurred |
| `cash_type` | Payment method (cash or card) |
| `card` | Anonymized card identifier |
| `money` | Transaction amount (ZAR) |
| `coffee_name` | Product purchased |
| `Time_of_Day` | Morning / Afternoon / Night |
| `Weekday` | Day of the week |
| `Month_name` | Month of the transaction |

**Products covered:** Americano, Americano with Milk, Cappuccino, Cocoa, Cortado, Espresso, Hot Chocolate, Latte

---

## 📊 Analytics & Insights

The analysis delivers insights across the following areas:

- **Sales Trends** — Daily, monthly, and hourly transaction volumes
- **Product Performance** — Best and worst performing coffee products
- **Revenue Analysis** — Total and average revenue by product, time, and period
- **Customer Behavior** — Peak hours, busiest days, and payment method breakdown
- **KPI Monitoring** — Transaction count, average order value, revenue growth
